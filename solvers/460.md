# An Ant on the Move

## Main Idea

The travel time along any straight segment equals the line integral of `1 / y`, so
vertical moves have cost `|ln(y1) - ln(y0)|`. Allowing vertical moves is essential:
the optimal path can start by moving straight up, then proceed with diagonal
segments, and finally move straight down.

The path is symmetric around `x = d/2`, so we compute only the left half and
double the result. The continuous optimal curve is a semicircle; the discrete
optimal path stays close to it. We therefore only keep lattice points within a
small band around the semicircle and run a dynamic program over that band.

## Algorithm Sketch

- Build a band of lattice points around the semicircle `(x - d/2)^2 + y^2 = (d/2)^2 + 1`.
- DP over `x = 0 .. d/2`, storing best time to each `y` in the band.
- For each `x`, relax vertical moves (adjacent `y` only) using two passes.
- Relax forward moves to `x + dx` for `dx` up to a safe bound.
- Answer is twice the minimum DP value at `x = d/2`.

## Notes

- `dx` is bounded empirically by `O(sqrt(d))`, which keeps the DP fast.
- The band width is small and constant, ensuring the state space stays compact.
