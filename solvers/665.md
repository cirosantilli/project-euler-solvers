# Project Euler 665 — Proportionate Nim (Solution Overview)

This problem studies a 2-pile impartial game where a move consists of choosing `n>0` and:

- removing `n` from one pile,
- removing `n` from both piles,
- removing `n` from one and `2n` from the other.

A **losing position** (P-position) is one where the player to move cannot force a win.

---

## Key technique: invariant-diagonal exclusion (GDWN)

This is exactly the game known as **(1,2)-Generalized Diagonal Wythoff Nim**.  
The losing positions can be generated greedily just like Wythoff pairs, but with **three diagonal invariants**:

For a candidate losing pair `(a,b)` (assume `a<b`), we require that the following quantities have **never occurred** in previous losing pairs:

- `d = b - a`  (slope-1 diagonal)
- `e = b - 2a` (slope-2 diagonal)
- `f = 2b - a` (slope-1/2 diagonal)

…and also their symmetric counterparts (because `(a,b)` and `(b,a)` are the same position).

So each new losing position is the smallest unused `a`, paired with the smallest feasible `b`.

---

## Data structures

To make the greedy generation efficient:

### 1. DSU / "next unused" structure
We maintain a disjoint-set union (union-find) so that:
- `find(x)` returns the next unused integer ≥ x
- marking a number used is done by linking it to `find(x+1)`

This prevents linear scanning through already-used values.

### 2. Bit arrays for invariants
Each invariant set is stored in a `bytearray` (Python) / `vector<uint8_t>` (C++), indexed with an offset so negatives are supported.

---

## Why we embed C++

The number of losing positions up to `M=10^7` is several million, and the greedy method requires constant-time checking and fast iteration.

Pure Python becomes too slow at this scale, so the final `main.py`:

- validates the sample assertions using Python
- compiles and runs an optimized C++ implementation (`-O3`)
- prints the exact answer for `f(10^7)`

---

## Complexity

- **Time:** ~O(number of losing positions)  
- **Memory:** about 150–200 MB due to DSU + invariant arrays
