# Project Euler 245 — Coresilience (Solution Summary)

We want all **composite** integers  
\[
1 < n \le 2\times 10^{11}
\]
such that the **coresilience**
\[
C(n)=\frac{n-\varphi(n)}{n-1}
\]
is a **unit fraction** \(1/k\).

---

## Key Mathematical Reductions

### 1) Unit fraction condition
\[
\frac{n-\varphi(n)}{n-1}=\frac1k
\quad\Longleftrightarrow\quad
k(n-\varphi(n)) = n-1
\]
Rearranging:
\[
k\varphi(n) - (k-1)n = 1
\]

### 2) Squarefree requirement
The equation above implies:
\[
\gcd(\varphi(n), n)=1
\]
But for any \(n\), \(\gcd(\varphi(n),n)\) contains all repeated prime factors of \(n\).  
Therefore **any solution must be squarefree**:
\[
n=\prod p_i
\quad\text{(distinct primes)}
\]

Thus we only search squarefree composites.

---

## Efficient Enumeration

### A) Semiprime case (\(n=pq\))
For \(n=pq\),
\[
\varphi(pq)=(p-1)(q-1)
\]
and the unit fraction condition becomes:
\[
(p-k)(q-k)=k^2-k+1
\]
So for each \(k\), let:
\[
S(k)=k^2-k+1
\]
Every divisor pair \(d\cdot e=S(k)\) gives:
\[
p=k+d,\qquad q=k+e
\]
We test whether both are prime and \(pq\le N\).

#### Fast factoring of \(S(k)\)
Instead of trial-dividing each \(S(k)\), we exploit that \(S(k)\) is the 6th cyclotomic polynomial:
\[
S(k)=\Phi_6(k)
\]
A prime \(p\) divides \(S(k)\) **iff** \(k\) is a primitive 6th root of unity mod \(p\), which happens only when:
- \(p=3\), or
- \(p\equiv 1\pmod 6\)

For each such prime \(p\), we compute the two roots of \(S(k)\equiv 0\pmod p\) and sieve across all even \(k\) values.

This yields a full factorization of all \(S(k)\) for \(k\le\sqrt N\) in well under a second.

---

### B) Multiprime case (\(\ge 3\) prime factors)
If \(n\) has at least 3 primes, then since all primes are \(>k\):
\[
n > (k+1)^3
\]
So:
\[
k \le \sqrt[3]{N} \approx 5848
\]
Thus the multiprime search space is *tiny*.

A depth-first search tracks partial products:
- \(P = \prod p_i\)
- \(A = \prod (p_i-1)\)

If the current prefix is missing only one prime \(r\), the defining equation becomes linear in \(r\), giving:
\[
r = \frac{kA + 1}{kA - (k-1)P}
\]
So the last prime is uniquely determined and easily checked.

---

## Implementation Techniques

- **Sieve of Eratosthenes** for all primes up to \(\sqrt N\)
- **Sieve-based factorization** of \(\Phi_6(k)\) values using modular roots
- **Deterministic Miller–Rabin** primality test (safe for the full search range)
- **DFS pruning** for multiprime solutions using:
  - \(k \le \sqrt[3]{N}\)
  - growth bounds on product \(P\)
  - positivity constraint \(kA-(k-1)P>0\)
