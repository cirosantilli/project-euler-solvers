# Project Euler 584 — Solution Summary

This problem asks for the **expected number of people** entering a room until we first find:

- **WimWi:** 3 birthdays within 1 day (year = 10 days)
- **Joka:** 3 birthdays within 7 days (year = 100 days)
- **Earth:** 4 birthdays within 7 days (year = 365 days)

Wrap-around counts (e.g., Dec 31 and Jan 1 are 1 day apart).

---

## 1) Rewriting the stopping-time expectation

Let `T` be the first time the condition is met, and let

- `p_n = P(T > n)` = probability we **have not yet** found a qualifying cluster after `n` people.

A standard identity for nonnegative integer stopping times is:

\[
\mathbb{E}[T] = \sum_{n\ge 0} p_n
\]

So we must compute `p_n` (or equivalently a way to sum them).

---

## 2) Exponential generating function trick (Poissonization)

Define the exponential generating function:

\[
G(t) = \sum_{n\ge 0} p_n \frac{t^n}{n!}
\]

Then:

\[
\mathbb{E}[T]
 = \sum_{n\ge0} p_n
 = \int_0^\infty e^{-t} \left(\sum_{n\ge0} p_n\frac{t^n}{n!}\right)\,dt
 = \int_0^\infty e^{-t}G(t)\,dt
\]

This works because:

\[
\int_0^\infty e^{-t}\frac{t^n}{n!}dt = 1
\]

So the problem becomes: **compute `G(t)` for a given `t` and integrate it**.

---

## 3) Turning `G(t)` into a finite-state automaton / transfer matrix

For a year of `N` days, let each day `i` have count `c_i` birthdays.

For Earth (K=4, D=7):

- “4 birthdays within 7 days” means **4 birthdays inside an 8-day window**.
- The complement event is:  
  **every length-8 window has at most 3 birthdays**.

That implies:

- Each day must have at most 3 birthdays.
- Every consecutive **7 days** also has sum ≤ 3.

Therefore, we can define a state as:

- the counts of the last **7 days** (since window length is 8),
- and only states with total sum ≤ 3 exist.

Number of such 7-tuples is:

\[
\binom{3+7}{7} = 120
\]

So Earth reduces to a **120-state automaton**.

A transition corresponds to choosing the next day's count `c` such that:

\[
\text{sum(last 7)} + c \le 3
\]

---

## 4) Weighting transitions (the EGF)

From multinomial reasoning, the generating weight of choosing count `c` for one day is:

\[
w_c(t) = \frac{(t/N)^c}{c!}
\]

So `G(t)` becomes the total weight of all valid cyclic sequences of length `N`
(which is computed as `trace(M(t)^N)`).

---

## 5) Integration using Gauss–Laguerre quadrature

We need:

\[
\mathbb{E}[T] = \int_0^\infty e^{-t}G(t)\,dt
\]

Gauss–Laguerre quadrature is specifically designed for integrals of the form:

\[
\int_0^\infty e^{-t} f(t)\,dt \approx \sum_i w_i f(x_i)
\]

So we compute:

\[
\mathbb{E}[T] \approx \sum_i w_i G(x_i)
\]

Nodes and weights are computed by finding roots of Laguerre polynomials,
using Newton refinement (Numerical Recipes method).

---

## 6) Validating with the sample answers

The problem statement provides:

- WimWi: `5.78688636`
- Joka:  `8.48967364`

The solution asserts these exact rounded values.

---

## 7) Output

Finally, we compute Earth:

- `N = 365`
- `D = 7` → window length 8
- `K = 4`

and print the expected value rounded to 8 decimals.
