# Least Common Multiple Count

## Key Ideas

- For each `n`, the number of pairs `(x, y)` with `x <= y` and `lcm(x, y) = n` is
  `f(n) = (1 + sum_{d|n} 2^{omega(d)}) / 2`, where `omega(d)` is the number of
  distinct prime factors.
- Let `h(n) = 2^{omega(n)}`. The summatory function is:
  `g(n) = n/2 + (1/2) * sum_{d<=n} h(d) * floor(n/d)`.
  This reduces the problem to a divisor-summatory over `h`.
- `h(n)` counts squarefree divisors, so its prefix sum can be computed via a
  multiplicative recursion: numbers are grouped by their smallest prime factor,
  which leads to a recursive sum over prime powers.
- A fast prime-counting function (Lehmer) provides the required prime counts in
  recursion base cases, making the approach efficient for `n = 10^12`.

## Implementation Notes

- Precompute primes up to `sqrt(n)` with a sieve.
- Precompute the prefix of `h(n)` up to `sqrt(n)` for quick small lookups.
- Use Lehmer prime counting for `pi(x)` to handle larger `x` during recursion.
- Use harmonic grouping for the outer divisor-summatory to reduce the number of
  distinct `floor(n/d)` values.

The program asserts the provided check `g(10^6) = 37429395` and prints
`g(10^12)`.
