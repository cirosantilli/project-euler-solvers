# Project Euler 419 — Look and Say Sequence

This solution computes:

- `A(n)`: number of `1` digits in the *n*-th look-and-say term  
- `B(n)`: number of `2` digits  
- `C(n)`: number of `3` digits  

for `n = 10^12`, with the final answer taken modulo `2^30`.

## Key techniques

### 1) Conway’s “audioactive decay” (92 atomic elements)

Conway proved that (after a few steps) any standard look-and-say sequence over digits `{1,2,3}` **splits into a concatenation of 92 fixed “atomic elements”** that then evolve independently.

Each element has:

- a concrete digit string (its “sequence”)
- a fixed **decay** into a (short) list of other elements after one look-and-say step
