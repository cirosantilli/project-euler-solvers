# Project Euler 524 — First Sort II

This repository solves Project Euler Problem 524 without any external libraries.

---

## Key techniques used

### 1) Understanding F(P)

The sorting process repeatedly finds the first inversion and moves the smaller
element to the front.

For a permutation `P` of `{1..n}`, the problem defines `F(P)` as the number of
steps until the list is sorted.

Instead of simulating the sorting, we use the known closed-form evaluation:

- Scan left to right.
- Maintain the maximum value seen so far.
- If the current value is **not** a new maximum, then it contributes a power of 2:

> contribution = 2^r where `r` is the number of previous elements smaller than it.

Then `F(P)` is the sum of all such contributions.

---

### 2) Removing powers of 2 from k

If `k` is even, the lexicographically first permutation with `F(P)=k`
always starts with `1`.

That means:

> **Q(n, k) = Q(n-1, k/2)** whenever `k` is even.

So we can repeatedly divide `k` by 2 and decrease `n` without changing the rank.

For the real problem input `k=12^12`, this removes `2^24` instantly,
shrinking the search from `45` elements down to `21`.

---

### 3) Lexicographically-first permutation via DFS + memoization

We need the lexicographically first permutation with a specific `F` value.

We build the permutation left-to-right:

- Try candidate values in increasing order (to guarantee lexicographic minimality).
- Track used values as a bitmask.
- Compute the next contribution in O(1) using `bit_count()`.
- Use memoization to avoid revisiting failed states `(mask, remaining_k)`.

Because the reduced instance is only size `21`, this search finishes quickly.

---

### 4) Computing the rank (Q) with factorials

Once we have the lexicographically first permutation, we compute its
1-indexed lexicographic rank using the factoradic method:

> rank = 1 + Σ (count_smaller_remaining * (n-i-1)!)

Again we use a bitmask + `bit_count()` to count unused values below a given element.

---

## Result

The program prints:

> `R(12^12) = 2432925835413407847`

which matches the known Project Euler answer.
