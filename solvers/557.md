# Project Euler 557 — Cutting Triangles (Solution Summary)

This repository contains a pure-Python solution for Project Euler Problem 557.

## Key techniques used

### 1) Affine coordinate model of the cut
Place the original triangle at `(0,0), (1,0), (0,1)` (area `1/2`).
Let one cevian start at vertex `B` and hit the opposite edge `AC`,
and another start at `C` and hit `AB`. Their intersection partitions
the triangle into:

- three triangles (areas `a,b,c`)
- one quadrilateral (area `d`)

Because area ratios are preserved under affine transforms, this model
captures all possible cuts.

### 2) Rational parameterisation of the cut
Represent where the cut hits each side using reduced fractions:

- `s = u/(u+v)` (point on `AB`)
- `t = p/(p+q)` (point on `AC`)

where `(u,v)=1` and `(p,q)=1`.

From the affine area formulas, every sub-area is a rational multiple
of the whole triangle area `T`. Therefore each valid configuration is
fully determined by these two reduced fractions plus a scaling factor.

### 3) Integer scaling via gcd/lcm
The areas have denominators involving:

- `K = u*q + v*p + v*q`
- `U = u+v`
- `P = p+q`

Instead of guessing a scale, we compute the **minimal** scale that makes
all four areas integers by forcing integrality of two of them (`b` and `c`)
using only:

- `gcd()` to cancel unavoidable shared factors
- `lcm()` to find the minimal scale satisfying both divisibility constraints

This yields a **primitive** quadruple `(a,b,c,d)` with total area `T0`.

### 4) Counting multiples efficiently
If `(a,b,c,d)` is primitive with total `T0`, then every multiple

`k*(a,b,c,d)` is also valid (total `k*T0`).

For each primitive total `T0 <= n`, the contribution to `S(n)` is:

`T0 * (1 + 2 + ... + m)` where `m = floor(n/T0)`

which is:

`T0 * m*(m+1)/2`

### 5) Deduplication
Different parameter choices can generate identical quadruples (especially
due to symmetry of swapping the two cut vertices). A `set()` of tuples
ensures each primitive quadruple is counted once.

## Included assertions (from problem statement)

- `S(20) = 259`
- For total area `55`, the only two quadruples are:
  - `(22, 8, 11, 14)`
  - `(20, 2, 24, 9)`

## Final result
Running `main.py` prints:

`2699929328`

which equals the known correct value for `S(10000)`.
