# Project Euler 591 — Best Approximations by Quadratic Integers

## Goal
For each non-square integer `d < 100`, find the quadratic integer

\[
a + b\sqrt{d}
\]

closest to \( \pi \), subject to:

\[
|a| \le n,\quad |b| \le n
\]

Then sum the absolute values of the integral parts \( |a| \).

---

## Key Techniques Used

### 1. Reduce to an Inhomogeneous Modular Approximation
For fixed `d`, define:

- \( \alpha = \{\sqrt d\} \)  (fractional part)
- \( \beta = \{\pi\} \)

Because `a` is an integer chosen nearest to \( \pi - b\sqrt d \), the error is:

\[
|\pi - (a + b\sqrt d)| = \min\left(|\{b\sqrt d\} - \beta|,\; 1-|\{b\sqrt d\}-\beta|\right)
\]

So we need a `b` such that the fractional part of \( b\alpha \) is very close to \( \beta \).

Negative `b` is handled by also searching the target \( 1-\beta \).

---

### 2. Use Continued Fractions of \( \sqrt d \)
The continued fraction of \( \sqrt d \) is periodic.  
If

\[
\sqrt d = [a_0; a_1, a_2, \dots]
\]

then

\[
\alpha = \sqrt d - a_0 = [0; a_1, a_2, \dots]
\]

Its convergents \( p_k/q_k \) provide denominators `q_k` such that:

\[
q_k \alpha \approx \text{integer}
\]

Thus the small remainder

\[
\delta = q_k\alpha - \text{round}(q_k\alpha)
\]

acts as a *tiny step size* modulo 1.

We choose `q` near \( \sqrt{B} \) (where \( B \approx n/\sqrt d \)) so the search is balanced.

---

### 3. Decompose `b = s*q + t`
Any \( b \le B \) can be written:

\[
b = s q + t,\quad 0 \le t < q
\]

Then:

\[
b\alpha \equiv t\alpha + s(q\alpha) \equiv t\alpha + s\delta \pmod 1
\]

Since \( \delta \) is very small, the number of modular wraps is tiny, so for each `t` we can solve
for the best `s` using only a *handful* of candidates.

Overall complexity per `d` becomes roughly **O(q)** where \( q \approx \sqrt{B} \sim 10^6\!-\!10^7 \),
and only ~90 values of `d` must be handled.

---

### 4. Fixed-Point Arithmetic (Power-of-Two Modulus)
To avoid floating-point cancellation (critical because errors are ~1e-13), we store fractional values
as integers modulo \( 2^{160} \).

That gives:

- Extremely fast modular arithmetic using bit masking (`& (MOD-1)`)
- Stability even for very tiny distances
