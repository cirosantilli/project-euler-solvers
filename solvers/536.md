# Project Euler 536 — Modulo Power Identity (Solution Summary)

## Problem
We want all integers `m ≤ n` such that

\[
a^{m+4} \equiv a \pmod m
\quad \text{for all integers } a
\]

Define `S(n)` as the sum of all such `m`.

---

## Key Number Theory Reductions

### 1) `m` must be squarefree
If `p^2 | m`, take `a = p`. Then:

- `a^{m+4}` is divisible by `p^2`
- but `a` is only divisible by `p`

So the congruence fails. Therefore **no prime square can divide `m`**.

---

### 2) Reduce to a condition on prime factors
For squarefree `m`, Chinese Remainder Theorem lets us check each prime divisor separately:

We need, for each prime `p|m`:

\[
a^{m+4} \equiv a \pmod p
\]

Over the field `F_p`, all nonzero `a` satisfy `a^{p-1}=1`.  
So we require:

\[
a^{m+3} \equiv 1 \pmod p
\quad \forall a \neq 0
\]

That holds **iff**:

\[
(p-1)\;|\;(m+3)
\]

Thus:

> **Valid `m` are exactly the squarefree integers such that**  
> **for every prime `p|m`, `(p-1)` divides `(m+3)`**.

Equivalently:

\[
\mathrm{lcm}(p-1\;:\;p|m)\;|\;(m+3)
\]

---

## Algorithmic Strategy

Brute force up to `10^12` is impossible, but solutions are *very sparse*.  
We use a **DFS over squarefree kernels** with strong congruence pruning.

### DFS State
Maintain:

- `A` = product of chosen primes
- `L` = `lcm(p-1)` over primes in `A`

Any completion `m = A·R` must satisfy:

\[
A\cdot R \equiv -3 \pmod L
\]

Let `g = gcd(A, L)`.  
This congruence is solvable iff `g | 3` (a powerful pruning rule).

Then we reduce the condition to one residue class:

\[
R \equiv T \pmod {L/g}
\]

When the number of `R` candidates in that arithmetic progression is small,
we **enumerate them directly**, factor them, and check the prime constraints.

---

## Important Techniques Used

- Sieve of Eratosthenes up to `sqrt(N)`
- Squarefree detection during factorization
- `lcm` tracking of `(p-1)` values
- CRT-inspired modular pruning via congruence classes
- Modular inverse via `pow(a, -1, mod)` (fast CPython implementation)
- DFS branching controlled by an enumeration threshold
