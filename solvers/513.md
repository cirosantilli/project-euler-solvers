# Integral Median

## Idea
Let the triangle sides be `a <= b <= c` and the median from `C` to `AB` be `m`.
The median formula gives:

```
m^2 = (2a^2 + 2b^2 - c^2) / 4
```

Thus `a` and `b` must have the same parity, and `c` is even. Set:

```
x = (a + b) / 2
y = (b - a) / 2
c = 2k
```

Then the median condition becomes:

```
x^2 + y^2 = k^2 + m^2
```

This is an equal-sum-of-two-squares relation. Using the classical bilinear
parameterization on doubled variables gives:

```
X = pr + qs
K = pr - qs
Y = |ps - qr|
```

with `X, Y, K` even, and then:

```
a = (X - Y) / 2
b = (X + Y) / 2
c = K
```

The triangle constraints reduce to `K > 0`, `K <= n`, and `X + Y <= 2K`.

## Algorithm
Enumerate integer parameters `p, r, s` with bounds derived from:

```
K = pr - qs <= n
pr >= 3qs
```

For each `(p, r, s)`, compute the valid interval(s) for `q` based on the
sign of `ps - qr` (two linear cases). Each valid `(p, q, r, s)` yields a
candidate triangle `(a, b, c)`. Deduplicate triangles using a packed integer
key.

## Complexity
The enumeration uses the bounds `pr <= 3n/2` and `qs <= pr/3` to keep loops
manageable. For the required `n = 100000`, the loop structure is intended to
run in a few seconds on PyPy.
