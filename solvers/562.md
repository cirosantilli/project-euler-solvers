# Project Euler 562 — Maximal perimeter

This repo contains a standalone Python solution (no external libraries) for Project Euler **562**.

## Core ideas

### 1) “No other lattice points” ⇔ primitive lattice triangle
If a triangle has integer-coordinate vertices and contains **no other lattice point** inside or on its edges, then by **Pick’s theorem** its area must be:

\[
A = i + \frac{b}{2} - 1
\]

With \(i=0\) and \(b=3\) (only the 3 vertices on the boundary), we get \(A=\tfrac12\).

Equivalently, after translating one vertex to the origin, the two edge vectors \(u, v\in\mathbb{Z}^2\) satisfy:

\[
|\det(u, v)| = 1
\]

That’s the “unimodular” / “primitive” condition.

### 2) Parameterizing the third vertex with a Diophantine equation
Fix two vertices \(A\) and \(B\), and let:

\[
d = A - B = (d_x, d_y)
\]

To get area \(1/2\), the third vertex must satisfy:

\[
\det(d, C-B) = \pm 1
\]

This is a linear Diophantine equation:

\[
d_x y - d_y x = \pm 1
\]

A particular solution \((x_0,y_0)\) is obtained from the **extended gcd** of \(d_x\) and \(d_y\).
All solutions are:

\[
C = B + (v_0 + k d)\quad (k\in\mathbb{Z})
\]

(and similarly for the \(-1\) orientation by using \(-v_0\)).

### 3) Disk constraint becomes a quadratic in \(k\)
The point \(C(k)\) moves along a line. The condition \(\|C(k)\|\le r\) becomes:

\[
\|w + k d\|^2 \le r^2
\]

which is a quadratic inequality in \(k\), giving an integer interval \([k_{\min}, k_{\max}]\).

### 4) Max perimeter occurs at an endpoint
Along a line, the function “distance to a fixed point” is convex, so the perimeter

\[
P(k)=|AB|+|BC(k)|+|CA(k)|
\]

is convex in \(k\). A convex function achieves its maximum over a closed interval at an endpoint, so it’s enough to check the endpoint integers \(k_{\min}\) and \(k_{\max}\).

### 5) Enumerating circle lattice points via sums of two squares
To search efficiently, we anchor one vertex \(B\) on the circle \(x^2+y^2=r^2\).
All such points can be generated from the prime factorization of \(r\) using:

- **Cornacchia’s algorithm** to represent primes \(p\equiv 1 \pmod 4\) as \(a^2+b^2=p\)
- multiplicativity in the **Gaussian integers** to combine representations

We keep only “canonical” points \((x\ge y\ge 0)\) because reflections and swapping coordinates preserve the problem.

### 6) Exact rounding of \(T(r)\)
For a primitive triangle, area is \(1/2\), so circumradius:

\[
R = \frac{abc}{2}
\]

Using squared side lengths \(a^2,b^2,c^2\), we compute \(N=a^2b^2c^2\) and:

\[
T(r) = \frac{R}{r} = \frac{\sqrt{N}}{2r}
\]

Rounding to the nearest integer is done with integer comparisons (no floating-point ambiguity):

\[
\sqrt{N} \ge \left(q+\tfrac12\right)2r \iff 4N \ge (2q+1)^2(2r)^2
\]

## What to run

```bash
python3 main.py
```

By default it prints the required answer for \(r=10^7\). You can pass another radius:

```bash
python3 main.py 100
```

The script includes asserts for the statement’s check values \(T(10)\) and \(T(100)\).
