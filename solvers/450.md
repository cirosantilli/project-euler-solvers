# Hypocycloid and Lattice Points

## Main Ideas

- Use complex form of the hypocycloid: `z(t) = (R-r) e^{it} + r e^{-i(R-r)t/r}`. If `cos t` and `sin t` are rational, then `e^{it}` is a rational point on the unit circle.
- Write `R-r = g p`, `r = g q` with `g = gcd(R-r, r)` and `gcd(p, q) = 1`. Then all points scale by `g`, and we sum over all `g` with `g(p+q) <= N`.
- Rational points on the unit circle come from `a,b` with `gcd(a,b)=1`, `n=a^2+b^2`. For `n=1,2`, all points are generated by the 4th roots of unity and can be summed with number-theory counts.
- For `n >= 5`, denominators grow like `d0^p` (with `d0 = n` if `n` odd, `d0 = n/2` if `n` even), so only small `n` and small `p` contribute. We explicitly enumerate primitive `(a,b)` with `n <= 2*sqrt(N/3)`.
- The key sums over coprime `p,q` with `p+q=s` reduce to:
  - `phi(s)` for constant contributions.
  - `S_abs(s) = sum_{gcd(q,s)=1} |s-2q|` for parity/mod-4 dependent cases.
  - `S_abs(s)` is computed by a Mobius convolution using a closed form for `sum_{k=1}^{t-1} |t-2k|`.
- Final total uses `sum_{g=1}^M g = M(M+1)/2` with `M = floor(N/(d*(p+q)))` for each base point.

## Complexity

- `O(N log N)` to compute `phi`, `mu`, and `S_abs` up to `N`.
- Small extra work for enumerating `n >= 5` pairs; negligible compared to the main sieve.
- Works comfortably for `N = 10^6`.
